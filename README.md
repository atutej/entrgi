## Overview

EntRGi is a method for steering discrete diffusion language models at inference time using entropy-aware reward guidance. Our key insight is to dynamically interpolate between soft embeddings and hard tokens based on the model's entropy, balancing the fundamental tension between gradient accuracy and reward model reliability.
